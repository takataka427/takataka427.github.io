概要

本サイトではhtmlを書くにあたって使える要素を3つ説明します。

本サイトの注意事項

本サイトは日本大学の演習課題によるサイトです。記載されている内容が必ずしも正しいとは限りません。

    <li>要素

    説明

    <li>要素は説明したい内容を項目ごとに分けるときに使用します。<li>要素は<ol>要素、<ul>要素のいずれかと一緒に使用します。

    マークアップ例

    <ul> <li>私は学生です。</li> <li>私は文理学部に所属しています。</li> <li>私は情報科学科です。</li> </ul>

    動作結果

    • 私は学生です。
    • 私は文理学部に所属しています。
    • 私は情報科学科です。

    <ol>要素

    説明

    <ol>要素はサイトの説明において、項目を分けるときに使用します。<ol>要素は項目ごとに番号で順番を表すことができます。<ol>タグの中に<li>タグを入れて使用します。

    マークアップ例

    <ol> <li>私は学生です。</li> <li>私は文理学部に所属しています。</li> <li>私は情報科学科です。</li> </ol>

    動作結果

    1. 私は学生です。
    2. 私は文理学部に所属しています。
    3. 私は情報科学科です。

    <ul>要素

    説明

    <ul>要素はサイトの説明において、項目を分けるときに使用します。<ul>要素は項目ごとに記号で行を表せます。<ul>タグの中に<li>タグを入れて使用します。

    マークアップ例

    <ul> <li>私は学生です。</li> <ul> <li>私は文理学部に所属しています。</li> <ul> <li>私は情報科学科です。</li> </ul> </ul> </ul>

    動作結果

    • 私は学生です。
      • 私は文理学部に所属しています。
        • 私は情報科学科です。
